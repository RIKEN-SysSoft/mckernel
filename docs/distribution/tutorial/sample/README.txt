%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
(1) How to run
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%

  1) Install IHK/McKernel
  Follow the instructions in README.

  2) Boot McKernel
  Follow the instructions in doc/HOW_TO_USE.
  
  4) Make the executable files
  Perform the following command.
  $ make

  5) Run the executable files
  Perform the following command. Assume IHK/McKernel is installed
  under <prefix>.
  $ <prefix>/bin/mcexec <executable-name>
